# Outbound Sales Analytics: End-to-End Portfolio Project

## Overview

This project analyzes outbound sales performance across leads, outreach activity, meetings, and opportunity outcomes to understand how sales efforts influence funnel conversion and pipeline performance.

The analysis focuses on outreach effectiveness, follow-up impact, lead source quality, and geographic performance using Power BI dashboards built on a relational sales data model.

---

## Project Highlights

- Analyzed outbound sales performance across:
  - 854 leads
  - 308 responses
  - 62 meetings
  - 7 closed-won opportunities
- Built 4 Power BI dashboards covering:
  - Funnel performance
  - Outreach effectiveness
  - Lead source analysis
  - Pipeline insights
- Identified LinkedIn as the highest-performing outreach channel for response and meeting conversion
- Analyzed follow-up effectiveness and funnel conversion across outreach stages
- Built a relational sales data model connecting leads, outreach activity, meetings, and opportunity outcomes

# Business Objectives

- Measure lead-to-win funnel conversion
- Evaluate outreach channel effectiveness
- Analyze follow-up sequence impact
- Identify high-performing lead sources
- Understand geographic pipeline performance
- Support outbound sales optimization using data-driven insights

---

# Key Insights

- LinkedIn generated the highest response and meeting conversion rates
- Response rates improved across multi-step follow-up sequences
- Major funnel drop-off occurred between response and meeting stages
- Certain lead sources generated higher-quality opportunities despite lower volume
- The US market contributed the highest number of meetings and wins
- Delayed feedback and inactive follow-ups were major reasons for lost opportunities
